这是由于在 github 中，devel 是项目的默认分支。要解决此问题，这样操作：

  1. 进入 github 中该项目的 Settings 页面；
  2. 设置 Default Branch 为其他的分支（例如 master）；
  3. 重新执行删除远程分支命令。
